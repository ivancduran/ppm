Phalcon Package Manager
==================

This is a cool program to install libraries in Phalcon Php


Installation
-------------

To install

    npm -g install phalcon-pm


example listing:

	ppm list

return:
	
	+ test
	+ crypt


example install:

	ppm install crypt

return:
	
	Package: crypt
	----------------
	Installing
	----------------
	Name        : Crypt AES 256
	Version     : 0.5.0
	Author      : @ivancduran
	Description : Encrypt and descrypt menssages with AES 256bits
	----------------
	Complete!
